# SSMShop
为了学习SSM框架做的一个家具购买的网站，LayUI + SSM + mysql
# 介绍
使用SSM开发模式实现一个简易的网站，主要功能：
（1）买家可以注册、登录、修改个人信息、下订单。
（2）卖家完成商品信息管理、订单管理。
（3）买家能看到订单状态信息。
（4）数据库采用mysql，表字段根据情况自拟。
